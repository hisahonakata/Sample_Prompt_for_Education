# 推薦状の作成

生成AIに推薦状を作成してもらいます。

# ツール

Bing Chat, ChatGPT

# プロンプト

```
指導教員として、山田太郎という学生に対する入社する会社への推薦状を書いてください。
彼の卓越した社会的、学問的、職業的能力を概説してください。
```
