# 短文形式のテスト問題作成

短文形式(短答式)のテスト問題を生成AIで作成します。

# ツール

Bing Chat, ChatGPT

# プロンプト

```
短文形式のテスト問題を{問題数}作成してください。
{テスト内容}に関するテスト問題を作成してください。
簡単な問題、中程度の問題、難しい問題を含めましょう。
多肢選択問題ではないことを確認してください。
正解は1つです。
それぞれの答えに合わせた解答を提供してください。

{問題数}:
6問

{テスト内容}:
デジタル学習デザイン
```
