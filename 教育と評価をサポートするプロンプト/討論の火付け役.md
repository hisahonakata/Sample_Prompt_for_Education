# 討論の火付け役

学生達が討論を行うための話題を生成AIに作成させます。

# ツール

Bing Chat, ChatGPT

# プロンプト

```
これからは学生達が討議を行います。
文化的、言語的に多彩なニーズを持つ{学年}に{テーマ}について、質問・問題・トピックを5つ作成してください。
批判的に考え、「賛成」と「反対」の答えを述べなさい。
クラスでの討論のために、裏付けとなる事例を毎回必ず含めてください。

{学年}:
大学1年生

{テーマ}:
アファーマティブ・アクション
```
